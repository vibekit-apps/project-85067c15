## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Sections

- **Hero Section**: Eye-catching header with animated candy icons
- **Product Grid**: Display of 8 featured candy products
- **Contact Form**: Complete contact form with store information
- **Header**: Navigation with shopping cart
- **Footer**: Links, social media, and newsletter signup

## Deployment

This project is ready to deploy to Vercel:

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy with zero configuration needed

## Customization

The design uses a custom color palette defined in `tailwind.config.ts`:
- `candy-pink`: #FF6B9D
- `candy-blue`: #4ECDC4
- `candy-yellow`: #FFE66D
- `candy-purple`: #A8E6CF

Modify these colors or add new ones to match your brand.
